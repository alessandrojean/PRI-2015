Introdução ao HTML

O HTML (HyperText Markup Language), é uma linguagem voltada para criação e formatação de página WEB.

  • Hyper é o oposto de linear. Nos termos antigos quando o mouse era apenas um rato caçado por gatos, os programas de computadores rodavam linearmente. Quando um programa executava uma ação, tinha que esperar a próxima linha de comando para executar a próxima ação uma características dos primeiros sistemas monotarefa ou monogramáveis como eram chamados. Com o HTML as coisas são diferentes, você pode ir de onde estiver para onde quiser. Exemplificando, não é necessário que você tenha visitando um determinado site antes de ter acessado outro.
  • Text nada mais é do que o texto que você desenvolve na forma de conteúdo para o internauta. Você também pode adicionar vários recursos como por exemplo: imagens. links, vídeos, banners, entre outros.
  • Markup significa que o texto que você escreve está dentro de tags de formatação que serão interpretadas pelo navegador. Você cria a marcação da mesma forma com que escreve em um editor de texto com definição de fonte, estilo, etc.
  • Language significa linguagem e é exatamente o que o HTML é, uma linguagem com várias tags de formação. Embora nos dias atuais os desenvolvedores WEB utilizam folha de estilo que facilita a formatação da página, além de restringir o acesso ao código.
